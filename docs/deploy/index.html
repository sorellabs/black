Black

Extensions to the base objects from the core JS library.

Overview

Black —at a glance—

Black Provides functionality missing from the core JavaScript library, for the basic object types, mostly pure and functional style.

You can grab the lastest release, or clone the entire Github repository:

Why Use It

While JavaScript could be regarded as a functional language, it lacks lots of utilities to deal with objects in a functional style. Black attempts to fix it by providing functions to deal with sequences, objects and other core objects the functional way.

Supported platforms

Black should work in any ECMAScript 5-compliant environment — like Node.js, Chrome and Firefox. Just load it and hack away.
